## ğŸ’¡ Project Description

This project implements a functional, standalone **Lightbox Viewer**, a common UI pattern used to display images in a centralized modal overlay while dimming the rest of the page content.

Users can click on any thumbnail image in the gallery to trigger the lightbox, displaying the full-sized image with controls to close the viewer either via a close button or by clicking outside the image.
